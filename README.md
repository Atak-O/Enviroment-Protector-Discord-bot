# 🌍 Environment Protector Discord Bot  
**Created by Atak-O (Atakan Özden)**  

> A Discord bot designed to promote environmental awareness and sustainable habits through interactive commands.

---

## 🚀 Features

- ♻️ **Recycling Guide (`!recycle`)**  
  Learn which waste goes into which recycling bin with a clean embed system.

- 🌱 **Environmental Advice (`!environmental_advice`)**  
  Get random eco-friendly tips to improve your daily habits.

- 🔐 **Password Generator (Utility Function)**  
  Includes a secure password generator function for future features.

---

## 🛠️ Technologies Used

- Python 🐍  
- discord.py  
- Discord Embed System  
- Built-in libraries (`random`, `string`)
